# Задание 1
Рассматривается множество целых чисел, принадлежащих отрезку [1012; 9638], которые делятся на 7 и оканчиваются на 1. Найдите сумму таких чисел.
~~~
t=0
for x in range(1012, 9639):
    if (x % 7 == 0) and (x % 10 == 1):
        t+=x
print(t)
~~~
# Задание 2
Рассматривается множество целых чисел, принадлежащих отрезку [1125; 7856], которые делятся на 3 и не делятся на 7 и 5. Найдите количество таких чисел.
~~~
t=0
for x in range(1125, 7857):
    if x % 3 == 0 and x % 7 != 0 and x % 5 != 0:
        t+=1
print(t)
~~~
# Задание 3
Рассматривается множество целых чисел, принадлежащих отрезку [100; 999], которые являются палиндромом (одинаково читаются слева направо и справа налево). Найдите сумму таких чисел.
~~~
t=0
for x in range(100, 1000):
    s = str(x)
    if s == s[::-1]:
        t+=x
print(t)
~~~
# Задание 4
В программу вводится два натуральных числа: N и K. Выведите на экран последовательность, состоящую из N чисел с разностью K (последовательность начинается с единицы).
~~~
N = int(input("Введите N: "))

# Задание 7
В программу вводится число N, после чего - N натуральных чисел, не превышающих 1000. Программа должна найти и вывести на экран максимальное число, кратное 7 и не кратное 9.
~~~
N = int(input("Введите N: "))
max_num = None
for _ in range(N):
    num = int(input())
    (if num % 7 == 0) and (num % 9 != 0):
        if max_num is None or num > max_num:
            max_num = num
if max_num is not None:
    print(max_num)
else:
    print("Таких чисел нет")
~~~
# Задание 8

# Задание 9
В программу вводится число N. Программа должна вывести два столбца чисел. В первом столбце – числа от 1 до N, во втором – числа от N до 1.

# Задание 10
В программу вводится число N. Требуется определить, является ли это число простым.
~~~
N = int(input("Введите N: "))
if N < 2:
    print("Не простое")
else:
    is_prime = True
    for i in range(2, int(N**0.5) + 1):
        if N % i == 0:
            is_prime = False
            break
    print("Простое" if is_prime else "Составное")
~~~
# Задание 11
В программу вводится натуральное число N. Требуется вывести на экран ряд чисел 1, 0.5, 0.25, 0.125, 0.0625, …, который содержит N элементов.
~~~
N = int(input("Введите N: "))
value = 1.0
for _ in range(N):
    print(value, end=" ")
    value /= 2
print()
~~~
